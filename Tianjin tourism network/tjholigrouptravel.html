推荐行程

温馨旅途：特别选用优质服务航空公司——韩亚航空，令您的旅途倍感温馨；

舒适行程：全程选用4花标准酒店，旅程全面升级；

内容丰富：包含景福宫、青瓦台、汉江游船、乐天世界通票、登首尔塔俯瞰整个首尔市容；

魅力购物：尽享乐天免税店购物乐趣，感受韩国时尚魅力；

特别安排：自由活动时间充裕，尽情体验首尔风情。

费用包含

国际往返机票及机场税、团队签证费、韩国四花级酒店住宿、景点首道大门票、行程中标明的餐食、全程中文导游及领队服务，赠送自费项目，旅游意外险

推荐行程

个人消费，推荐自费项目 ，导游小费

自费项目

无

接待标准

无

友情提示

1.基本购物站如客人要求取消，每减少一个购物站须交纳团费人民币 300元/人，非购物团除外。

2.客人在旅游途中，如有私自离团，须交纳人民币1000元/人、天，以一整天作为基准，基本收费单位为一天，离团期间团费无退还。

3.如发生单男单女，加房须交纳人民币 价格待订

4. 团中有外籍人士，朝鲜族，团费价位另计。

5. 本公司一律不接受留学生参团，如参团团费另计。

6. 本公司一律不接受境外参团，如参团团费另计。

7. 本公司一律不接收同行人士，如参团团费另计。

第1天天津-青州(飞机)

行程：请您提前抵达北京首都机场三号航站楼集合，在领队的协助下办理登机、出关手续，随后准备乘坐中国国际航空公司客机飞往动感活力的时尚之都—首尔，抵达仁川机场后，中文导游接机，午餐后，赴清溪川（不少于60分钟），感受时尚魅力，演绎新首尔物语。在清溪川，感受贴近自然的原生态环境，随后安排搭乘汉江游船，欣赏完江边夜景后用晚餐，之后返回酒店休息；

  • 活动区域：出境
  • 用餐信息：无
  • 住宿信息： 四花酒店

  • 天津国滨海际机场集合赴韩国清州。抵达后，安排住宿。
